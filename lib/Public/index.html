Twin Peaks API

How to get quotes:

  • Get All Quotes:
    https://damp-cove-34137.herokuapp.com/api/quotes/
  • Get Random Quote:
    https://damp-cove-34137.herokuapp.com/api/quotes/random
  • Get Quotes By Id:
    https://damp-cove-34137.herokuapp.com/api/quotes/[id]
    https://damp-cove-34137.herokuapp.com/api/quotes/3
  • Get Quotes By Character:
    https://damp-cove-34137.herokuapp.com/api/quotes/characters/[Name]
    https://damp-cove-34137.herokuapp.com/api/quotes/characters/Dale Cooper
  • Get Quotes By Search Term:
    https://damp-cove-34137.herokuapp.com/api/quotes/search/[term]
    https://damp-cove-34137.herokuapp.com/api/quotes/search/owls
  • Limit Number Of Quotes:
    https://damp-cove-34137.herokuapp.com/api/quotes/limit/[count]
    https://damp-cove-34137.herokuapp.com/api/quotes/limit/5
  • Get Random Quote From Specific Character:
    https://damp-cove-34137.herokuapp.com/api/quotes/characters/random/[Name]
    https://damp-cove-34137.herokuapp.com/api/quotes/characters/random/Gordon Cole
  • Get Specific Character Information:
    https://damp-cove-34137.herokuapp.com/api/characters/[name]
    https://damp-cove-34137.herokuapp.com/api/characters/Log Lady
  • Gets All Character Information:
    https://damp-cove-34137.herokuapp.com/api/characters/
